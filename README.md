# AI Career & Certification Advisor

## Overview
AI Career & Certification Advisor is a Creative Apps project for the Microsoft Agents League Hackathon 2026.

The app helps aspiring data professionals and career switchers understand their strengths, skill gaps, recommended certifications and project ideas.

## Problem
Career switchers often have certifications but do not know how to connect them to real job roles, project portfolios and interview preparation.

## Solution
This app allows users to enter their background and target career direction. It generates structured career advice, suggested next steps and CV-ready project ideas.

## Microsoft IQ Layer
This project uses a Foundry IQ-style grounded knowledge retrieval approach. The app references a local knowledge base to provide grounded, source-based recommendations and reduce unsupported answers.
