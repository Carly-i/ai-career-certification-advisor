# Career and Certification Knowledge Base

## PL-300 Power BI Data Analyst
PL-300 validates skills in preparing, modelling, visualising and analysing data using Microsoft Power BI. It is relevant for Data Analyst, BI Analyst and Reporting Analyst roles.

## AWS Certified Cloud Practitioner
AWS Cloud Practitioner validates foundational cloud knowledge, including cloud concepts, security, billing, pricing and core AWS services. It is useful for understanding cloud-based data platforms.

## Recommended next skills for Data Analyst
Common skills for entry-level Data Analyst roles include SQL, Excel, Power BI, data cleaning, dashboard design, business analysis and basic Python.

## Recommended next certifications
For a Power BI learner, Microsoft Fabric-related certifications such as DP-600 may be useful. For cloud administration, AZ-104 may be useful. For AI application development, AI-900 or AI-102 may be considered depending on experience level.

## Project portfolio advice
Strong beginner data projects should include a clear problem, dataset, analysis process, dashboard or AI-assisted insight, and business recommendations. A GitHub repository and demo video help make the project more interview-ready.
